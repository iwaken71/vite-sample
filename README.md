# 背景
Vite + Vue.js 3 + Babylon.jsのサンプルを目指しました。

# Setup

clone
``` 
git clone git@github.com:iwaken71/vite-vue-babylon-sample.git
cd vite-vue-babylon-sample
```

SetUp

```
npm install
```

Dev実行

```
npm run dev
```

Build
./dist以下にbundleファイルができる
```
npm run build
```
